☁ Cloud Sync (Supabase)

Free persistent storage — syncs across devices, survives clearing browser cache.
▶ One-time setup instructions
1. Go to supabase.com → Sign up free (GitHub login works)
2. New project → pick any region → wait ~2 min for it to provision
3. Project Settings → API → copy Project URL and anon public key
4. SQL Editor → New query → paste this and click Run:
create table if not exists ledger_data (
  user_id text not null,
  key text not null,
  value text,
  primary key (user_id, key)
); 5. Paste URL + key above → set a User ID → click Push
6. Done! Every change auto-syncs. On a new device: enter same credentials → Pull.
